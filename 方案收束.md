# 榴莲人格导购官：方案收束

## 一句话

用 SBTI 式人格测试作为低成本社交入口，把用户的口味、预算、风险接受度和黑暗品接受度转成可分享的人格卡，并沉淀为商家的导购、直播和非标果转化线索。

## 商业闭环

```text
年轻用户被测试吸引
→ 完成 10 题榴莲 SBTI
→ 获得人格结果、品种推荐、避坑建议
→ 生成分享卡，带来好友裂变
→ 商家后台沉淀购买意愿和偏好
→ 生成直播话术、拼团策略、黑暗品数字名片
→ 反哺转化和库存处理
```

## MVP 必做

1. 前台人格测试：10 题，每题都映射真实导购变量。
2. 结果页：人格、推荐品种、购买形式、避坑建议、购买意愿。
3. 分享卡：可生成图片，适合发朋友圈/群聊。
4. 商家后台：展示人格分布、购买意愿、分享潜力、黑暗品接受度。
5. 黑暗品数字名片：把裂口、丑果、刺不均匀等非标问题透明说明。

## 不做

1. 不做真实交易系统。
2. 不做复杂登录。
3. 不做真实库存系统。
4. 不做成熟度图像识别第一版。

## 答辩表达

这个项目不是单纯的整活测试。我们借 SBTI 的低门槛和截图传播机制，把年轻用户带进来；再用测试结果完成榴莲推荐和分享卡裂变；最后把每一次测试沉淀成商家的用户画像、直播话术和黑暗品销售线索。它前台好玩，后台能卖货。
